# Git
## <span class="gold">Command Line Basics</span>

#### A primer on navigating the Git command line

```shell
$ git add PITCHME.md
$ git commit -m "New slideshow content."
$ git push

Done!
```

@title[Getting Started]
### Getting Started
<br>
#### Debian
```shell
$ apt-get install git
```
<br>
#### CentOS
```shell
$ yum install git
```
<br>
#### MacOS
```shell
$ brew install git
```
